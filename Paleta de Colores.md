# Paleta de Colores Corporativa

Este documento describe la paleta de colores oficial y su uso recomendado para productos digitales, branding y comunicación visual. La paleta está pensada para ser flexible, accesible y consistente en interfaces web y móviles.

---

## 1. Color Principal

**Principal – Verde**
- Color base de la marca. Representa crecimiento, confianza y acción positiva.
- Uso recomendado: CTAs principales, elementos destacados, indicadores positivos.

| Variante | Hex |
|--------|-----|
| Principal | `#3BB54A` |
| Principal 80% | `#D6F2DA` |
| Principal 40% | `#85D78F` |
| Principal +25 | `#2C8837` |
| Principal +50 | `#1D5B25` |

---

## 2. Color Secundario

---

## 6. Escala de Neutros

- Base para tipografía, fondos y contenedores.
- Garantiza legibilidad y jerarquía visual.

| Uso | Hex |
|----|-----|
| Dark | `#222428` |
| Medium | `#92949C` |
| Light | `#F4F5F8` |

---

## 7. Recomendaciones Generales

- Usar **Principal** para acciones clave.
- Reservar **Danger** exclusivamente para errores reales.
- Mantener contraste AA/AAA en texto según WCAG.
- Neutros como base, colores como énfasis.
- Evitar usar más de 2 colores fuertes simultáneamente en una misma vista.
